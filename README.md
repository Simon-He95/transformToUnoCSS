# Install

```
  npm i -g transform-to-unocss
```

# cli

```
  转换：tounocss + 目录
  tounocss playground
  撤销： tounocss + 目录 + --revert
  tounocss payground --revert
```

# vite

```
  import { vitePluginTransformToUnocss } from 'transform-to-unocss'
  plugins: [ vitePluginTransformToUnocss() ]
```
